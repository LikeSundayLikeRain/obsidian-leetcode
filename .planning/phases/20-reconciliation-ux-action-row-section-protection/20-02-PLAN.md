## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Widget DOM → Plugin (button click) | User input crosses from the action-row button into `*FromWidget(widget)` plugin methods |
| Plugin → Frontmatter (chevron click → processFrontMatter) | Plugin writes to the user's note's frontmatter |
| Frontmatter → Widget (metadataCache.on('changed')) | Untrusted frontmatter content (could be anything in `lc-language`) crosses back to the widget's Compartment.reconfigure |
| External edit (vault.on('modify')) → Widget | OS file-system tools / Obsidian Sync / other panes modify the file on disk |
