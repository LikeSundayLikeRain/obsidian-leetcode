Purpose: Add a sibling predicate + repair path that recognizes the asymmetric "v1.3 body + missing lc-language" shape and injects `lc-language: <plugin.settings.getDefaultLanguage()>` via `processFrontMatter` BEFORE the widget mount path's Notice fires.

Reading-mode + Live Preview entry points invoke the repair alongside the existing migrate call. The widget's `resolveLanguageSlug` Notice path is unchanged (intentional — the repair runs BEFORE mount, so the Notice never fires on the repaired path; non-repaired no-slug-no-fence cases keep the existing fallback).
