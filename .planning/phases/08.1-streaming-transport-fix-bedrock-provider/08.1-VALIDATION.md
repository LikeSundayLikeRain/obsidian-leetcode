## Sampling Rate

- **After every task commit:** Run `npm test -- --run <touched test file>` + relevant grep gate.
- **After every plan wave:** Run `npm test -- --run` + `npm run build` + `npm run lint`.
- **Before `/gsd:verify-work`:** Full suite green; both grep gates green; bundle under ceiling.
- **Max feedback latency:** ≤ 30 s per task; ≤ 90 s per wave.
