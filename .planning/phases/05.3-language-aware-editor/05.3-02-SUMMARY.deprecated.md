3 source files (455 lines) implement the Wave-0 contract from Plan 01: per-view Compartment swap driven by caret-in-fence StateField, lazy `import()` Promise cache for the 7 first-party packs, and `indentService`-based whitespace-copy fallback. ### Task 1 (commit `fa0d0e7`) — `languagePackRegistry.ts` + `whitespaceCopyIndent.ts`

**`src/main/languagePackRegistry.ts` — 133 lines:**
- Exports `PackId` (7-member union: `python | java | cpp | javascript | typescript | go | rust`), `getLanguagePack`, `slugToPackId`, `warmDefaultPack`.
- `getLanguagePack(id)` backs onto module-level `Map<PackId, Promise<LanguageSupport>>`. First call triggers `loadPack(id)`; subsequent calls return the cached promise.
- On rejection, `cache.delete(id)` runs in the `.catch` so a retry can re-trigger the underlying dynamic import (RESEARCH lines 783–790).
- `loadPack(id)` switch body — exact match to RESEARCH lines 792–803:
  - `python` → `(await import('@codemirror/lang-python')).python()`
  - `java` → `(await import('@codemirror/lang-java')).java()`
  - `cpp` → `(await import('@codemirror/lang-cpp')).cpp()`
  - `javascript` → `(await import('@codemirror/lang-javascript')).javascript()`
  - `typescript` → `(await import('@codemirror/lang-javascript')).javascript({ typescript: true })`
  - `go` → `(await import('@codemirror/lang-go')).go()`
  - `rust` → `(await import('@codemirror/lang-rust')).rust()`
- Type-only import of `LanguageSupport` from `@codemirror/language` — keeps runtime zero-cost; no eslint suppression needed.
- `warmDefaultPack(slug)` resolves slug→PackId via `slugToPackId`, then fire-and-forget `void getLanguagePack(id).catch(() => {})` (silent CF-19).

**`src/main/whitespaceCopyIndent.ts` — 73 lines:**
- Exports `computeWhitespaceCopyIndent(state, pos): number` (named pure function for Wave-0 testability per PATTERNS Option A) + `whitespaceCopyIndentExtension: Extension`.
- Body matches RESEARCH lines 525–540: `line.number === 1 → 0`; walks back through empty lines; first non-empty preceding line's leading whitespace is extracted via `/^[\t ]*/`; tabs replaced by 2 spaces; column count returned.
- Extension is `indentService.of((ctx, pos) => computeWhitespaceCopyIndent(ctx.state as never, pos))`.
- 2 transitive-peer-dep eslint suppressions for runtime `indentService` and type-only `Extension` imports.

**Tests:** 27 GREEN (20 registry + 7 indent).

### Task 2 (commit `4b39f08`) — `codeFenceLanguageExtension.ts`

**`src/main/codeFenceLanguageExtension.ts` — 249 lines (≥140 required):**
- Exports `DesiredPack` type, `FenceState` interface, `EMPTY` sentinel (module-private), `mapSlugToPack(slug)`, `computeFenceState(state, plugin)`, `buildCodeFenceLanguageExtension(plugin)`.
- `mapSlugToPack` switch body matches RESEARCH lines 381–394: pragmatic 8 → respective PackId; default → `'fallback'`.
- `computeFenceState` is a pure function — 4 short-circuit gates:
  1. `editorInfoField.file` present.
  2. `metadataCache.getFileCache(file)?.frontmatter['lc-slug']` is a non-empty string.
  3. `findCodeFence(state)` (REUSED from 5.1 — D-13 scoping) returns a fence.
  4. `caretLine > openerLine && caretLine < closerLine` (strict-inside, RESEARCH Open Question 4).
- On success: extract opener-line text → `/^\s*```\s*(\S+)?/` → `resolveLangSlug(fenceTag, '')` → `mapSlugToPack`.
- `buildCodeFenceLanguageExtension(plugin)` returns `[fenceStateField, viewPlugin, languageCompartment.of([])]`. Each call creates a fresh `Compartment` per view (RESEARCH Anti-Pattern §"Storing the compartment globally").
- StateField `update` recomputes on `tr.docChanged || tr.selection || modeFlipped` (mode flip = `editorLivePreviewField` differs).
- ViewPlugin `update(vu)` guards `prev.desiredPack !== next.desiredPack` (RESEARCH Pitfall 5 — load-bearing).
- ViewPlugin constructor kicks an initial `void this.swap(s)` if mounted-state caret is in fence.
- `swap(target)` async:
  - `desiredPack === 'fallback'` → `whitespaceCopyIndentExtension`.
  - `desiredPack === PackId` → `await getLanguagePack(target.desiredPack)`; on `.catch`, silently fall back to `whitespaceCopyIndentExtension` (CF-19).
  - `desiredPack === 'none'` → empty array.
  - Guards `(this.view as EditorView & { destroyed?: boolean }).destroyed` before dispatch (RESEARCH Pitfall 3).
  - `view.dispatch({ effects: languageCompartment.reconfigure(ext) })`.
- Reuses `findCodeFence` (5.1), `resolveLangSlug` (5.0), `getLanguagePack` (Task 1), `whitespaceCopyIndentExtension` (Task 1) — every analog/contract from PATTERNS honored.

**Tests:** 26 GREEN (Wave-0 RED contract for codeFenceLanguageExtension transitions to GREEN). 5.1 `codeActionsEditorExtension.test.ts` continues to pass (12 tests) — D-16 regression confirmed.
