## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| AI response → vault write | Untrusted AI text written to user note via vault.process |
| AI response → modal DOM | Untrusted AI text rendered via MarkdownRenderer.render (CSP-safe) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-06 | Tampering | vault.process write | mitigate | mergeAIReviewSection is pure + idempotent; vault.process is atomic with retry |
| T-09-07 | Elevation of Privilege | AI response rendering | mitigate | MarkdownRenderer.render strips script tags; never use innerHTML |
| T-09-08 | Denial of Service | Cost abuse via repeated auto-reviews | accept | Cost ledger accumulates; user must opt-in; single review per AC event |
| T-09-09 | Information Disclosure | ## Notes leakage | mitigate | buildReviewPrompt never receives ## Notes content; locked by test assertion in Plan 01 |
