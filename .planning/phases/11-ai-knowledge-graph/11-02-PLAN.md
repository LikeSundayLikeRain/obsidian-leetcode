<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| AI response -> vault writes | AI-returned pattern names become file paths + wikilinks; must be normalized |
| AI response -> hub note content | AI-returned problem slugs become wikilinks in hub notes; validated against index |
| AI provider network -> plugin | Classification call may fail; never-throw posture prevents cascade |
| User input (OtherPatternModal) -> vault | User-typed pattern name normalized via normalizePatternName before use as file path |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-04 | Tampering | PatternClusterEngine | mitigate | normalizePatternName sanitizes AI output before it becomes a file path; no path traversal possible (Patterns/ prefix + basename only) |
| T-11-05 | Denial of Service | ClusterHubWriter.reconcile | mitigate | Filter vault scan by problemsFolder prefix; use metadataCache (in-memory) not file reads; bail early on empty results |
| T-11-06 | Elevation of Privilege | PatternClusterEngine | mitigate | AIClient DI boundary preserved — src/graph/ never imports obsidianFetch directly; LC-isolation test enforces this |
| T-11-07 | Denial of Service | PatternClusterEngine.onAccepted | mitigate | Cost tracked via addCostLedger; inherits daily cost cap when AIREV-06 ships; classification is non-streaming (fast, bounded) |
| T-11-08 | Tampering | OtherPatternModal user input | mitigate | User input normalized via normalizePatternName (trim, collapse whitespace, title-case); no raw user string used as path |
| T-11-SC | Tampering | npm/pip/cargo installs | accept | No new packages installed — turndown already in bundle |
</threat_model>
