### `src/solve/runArity.ts` (NEW — utility, pure transform)

**Analog:** `src/solve/statusMap.ts` (pure module, no I/O, no imports beyond types).

**Module shape pattern** (statusMap.ts:1-44 — entire file):
```typescript
// src/solve/statusMap.ts
//
// LC status_code integer → verdict kind + display name.
// Reference: skygragon/leetcode-cli lib/helper.js statusToName (2019).
// Phase 3 adds the 'unknown' fallback per CONTEXT D-15 ...
//
// Purity: no imports, no state, no I/O.

export type VerdictKind = 'ac' | 'wa' | ... | 'unknown';

export interface StatusInfo {
  kind: VerdictKind;
  displayName: string;
}

const KNOWN: Readonly<Record<number, StatusInfo>> = { /* ... */ };

export function classifyStatus(code: number, msg?: string): StatusInfo {
  const known = KNOWN[code];
  if (known) return known;
  return { kind: 'unknown', displayName: msg ?? `Unrecognized status ${code}` };
}
```

**Apply to runArity.ts:**
- File header comment block describing purity (no imports beyond types, no DOM, no I/O).
- Export named pure functions: `parseMetaData`, `deriveArity`, `splitInput`, `splitOutput` (or equivalent).
- Each function MUST handle malformed input via try/catch + sentinel return (D-08 raw-dump fallback). The `classifyStatus` "unknown fallback" pattern is the precedent — never throw on malformed input.

**Splitter precedent — `splitExampleTestcases`** (ephemeralTabStore.ts:147-153):
```typescript
/** Split LC's `exampleTestcases` payload into distinct tab inputs. LC uses a
 *  blank-line boundary between cases; per-case-internal newlines are
 *  preserved. Empty / whitespace-only input → empty list. */
export function splitExampleTestcases(raw: string): string[] {
  if (!raw) return [];
  return raw
    .split(/\n\s*\n/)
    .map((s) => s.trim())
    .filter((s) => s.length > 0);
}
```

**Apply to runArity.ts splitInput:** mirror this signature shape (`(raw: string, arity: number) => string[]`); leading `if (!raw) return []` defensive check; chained `.map(s => s.trim()).filter(Boolean)` style.

```typescript
function appendEl(parent: HTMLElement, tag: string, cls?: string): HTMLElement {
  const el = (parent.ownerDocument ?? activeDocument).createElement(tag);
  if (cls) el.className = cls;
  parent.appendChild(el);
  return el;
}

function setText(el: HTMLElement, text: string): void {
  // textContent is the safe equivalent of setText — string-as-text, never

}
```
