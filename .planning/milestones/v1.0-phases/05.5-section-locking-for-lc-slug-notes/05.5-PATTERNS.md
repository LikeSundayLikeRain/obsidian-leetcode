The detector (excerpt from analog lines 168-203) — DO NOT copy these into 05.5; just reference and import:

```typescript
export function findCodeFence(
  state: EditorState,
): { openerLine: number; closerLine: number } | null {
  if (state.doc.lines === 0) return null;
  const FENCE_RE = /^\s*```/;
  const H2_CODE_RE = /^\s*##\s+Code\s*$/;
  const H2_ANY_RE = /^\s*##\s+.+$/;
  let inCodeSection = false;
  const total = state.doc.lines;
  for (let i = 1; i <= total; i++) {
    const text = state.doc.line(i).text;
    if (H2_CODE_RE.test(text)) { inCodeSection = true; continue; }
    if (H2_ANY_RE.test(text)) { inCodeSection = false; continue; }
    if (inCodeSection && FENCE_RE.test(text)) {
      for (let j = i + 1; j <= total; j++) {
        if (FENCE_RE.test(state.doc.line(j).text)) {
          return { openerLine: i, closerLine: j };
        }
      }
      return null; // unterminated fence
    }
  }
  return null;
}
```

**D-09 fall-through behavior:** when `findCodeFence` returns `null` (unterminated / missing fence), the lock emits ONLY the `## Code` heading-line lock and lets the body fall through to "everything else editable." This matches Phase 5.1's null-return semantics (`buildDecorations` returns an empty set on null) but the new file uses null as a "skip the body lock for this section" signal instead.

The analog's structure is `buildCodeActionsEditorExtension(plugin) → Extension` returning a `StateField.define<DecorationSet>({ create, update, provide })`. 05.5 follows the **same outer signature** (`buildSectionLockExtension(plugin) → Extension`) but the inner shape is a composed array of facets, not a StateField:

// Pattern derived from RESEARCH §Pattern 2 + Pattern 3; outer signature mirrors Phase 5.1.
export function buildSectionLockExtension(plugin: Plugin): Extension {
  return [
    EditorState.changeFilter.of((tr) => {
      // 1. Pitfall 5 — let plugin-side dispatches through (chevron switch, etc.)
      const ev = tr.annotation(Transaction.userEvent);
      if (typeof ev === 'string' && ev.startsWith('leetcode.')) return true;
      // 2. Phase 5.1 frontmatter gate (verbatim shape; tr.startState NOT state)
      const file = tr.startState.field(editorInfoField)?.file;
      if (!file) return true;
      const fm = plugin.app.metadataCache.getFileCache(file)?.frontmatter as
        | Record<string, unknown> | undefined;
      if (typeof fm?.['lc-slug'] !== 'string') return true;
      // 3. Compute locked ranges — return [from, to, from, to, ...] suppression array
      const ranges = computeLockedRanges(tr.startState);
      return ranges.length === 0 ? true : ranges;
    }),
    EditorView.atomicRanges.of((view) => buildAtomicRangeSet(view.state)),
    // OPTIONAL — visual dim (planner discretion per CONTEXT D-04):
    // EditorView.decorations.of((view) => buildLockedDecorations(view.state)),
  ];
}
```
