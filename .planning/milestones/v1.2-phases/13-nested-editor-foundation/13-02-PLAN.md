  <action>
    Create `src/main/nestedEditorExtension.ts`. Structure:

    1. **PluginHost type** ŌĆö `Plugin & { childEditorRegistry: ChildEditorRegistry; app: App }` (structural typing, minimal).

    2. **NestedEditorWidget extends WidgetType** ŌĆö Constructor takes `filePath: string`, `registry: ChildEditorRegistry`, `fenceContent: string`. `eq(other)` compares ONLY `filePath` (per D-13, prevents rebuild on doc edits). `toDOM(view)` creates `div.lc-nested-editor`, calls `registry.get(filePath)` ŌĆö if hit, `container.appendChild(childView.dom)`; if miss, calls `createChildEditor(fenceContent, container)` then `registry.set(filePath, childView)`. `destroy(dom)` detaches child `.cm-editor` element from container but does NOT destroy the EditorView (D-13). `ignoreEvent()` returns false. `estimatedHeight` returns `Math.max(fenceContent.split('\n').length * 20, 60)`.

    3. **extractFenceBody(state, fence)** ŌĆö Returns `state.doc.sliceString(line(openerLine+1).from, line(closerLine-1).to)` for body between opener/closer. Returns empty string if closerLine - openerLine <= 1.

    4. **buildNestedDecorations(state, plugin)** ŌĆö Uses `RangeSetBuilder`. Gate: `editorInfoField` file check, then `metadataCache.getFileCache(file)?.frontmatter?.['lc-slug']` string check, then `findCodeFence(state)`. If all pass: iterate `fence.openerLine` to `fence.closerLine` adding `Decoration.line({ class: 'lc-fence-hidden' })` at each `line(i).from`. Then add `Decoration.widget({ widget: new NestedEditorWidget(...), block: true, side: 1 })` at `line(openerLine).to`. Return `builder.finish()`.

    5. **buildNestedEditorExtension(plugin): Extension** ŌĆö Returns array of `[stateField, transactionFilter]`. StateField: `create` calls `buildNestedDecorations`; `update` rebuilds on `tr.docChanged`, else `old.map(tr.changes)`; `provide` returns `EditorView.decorations.from(f)`. TransactionFilter: checks `tr.selection`, gates on `lc-slug`, finds fence, checks if `sel.main.head` is within `fenceFrom..fenceTo`, if so snaps cursor to `fenceFrom - 1` (if coming from above) or `fenceTo + 1` (if from below) and calls `queueMicrotask(() => childView.focus())`.

    Import `findCodeFence` from `./codeActionsEditorExtension`. Import `ChildEditorRegistry` from `./childEditorRegistry`. Import `createChildEditor` from `./childEditorFactory`. Import `editorInfoField` from `obsidian`. Use eslint-disable for all `@codemirror/*` imports.
